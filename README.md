## 📝 Sobre o Projeto
Este projeto faz parte do desafio **"Natural ou Fake Naty?"** da **DIO**. Como embaixadora e mãe de 4 filhos, utilizei ferramentas de IA para criar um avatar digital realista que comunica minha jornada no desenvolvimento diretamente pelo celular.

## 🎯 Objetivo
Desafiar a percepção entre o real e o gerado por IA com uma comunicação fluente em **Português (Brasil)**, utilizando avatares de alta fidelidade.

## 🛠️ Tecnologias Utilizadas
* **HeyGen:** Para a criação do avatar, geração de voz e sincronia labial (lip-sync).
* **CapCut:** Para a edição final, ajustes de vídeo e finalização.
* **YouTube Shorts:** Para hospedagem e divulgação do resultado.
